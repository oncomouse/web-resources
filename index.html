CSS font stacks are a way to better guarentee a viable web-safe font will be loaded. Copy & paste these into your CSS code for better font support.

Google Web Fonts

Google hosts one of the largest free repositories for loading web fonts into CSS. Web fonts are widely supported (though not in older browsers) and greatly increase the available fonts for use in web design. Google does not offer as high quality or as wide a selection as a paid service such as Typekit, however.
